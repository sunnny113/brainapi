First request

Unified endpoint example 
curl -X POST https://api.brainapi.site/api/v1/ai \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "text",
    "input": "Summarize the latest events from my app for a returning user.",
    "mode": "cheap"
  }'
